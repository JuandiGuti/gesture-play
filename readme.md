# Frontend
1. Instalar Python
2. Ir a la carpeta de frontend
```
cd frontend
```

3. Instalar una vm
```
python -m venv venv
```

4. Activar la vm
-Windows
```
venv\Scripts\activate
```

-Mac/Linux
```
source venv/bin/activate
```

5. Instalar flask
```
pip install flask
```

6. Activar el frontend
```
python app.py
```
